**En**

This plugin connects minecraft and telegram chat
when sending sms messages to telegram chat, they will be cloned in minecraft also from minecraft to telegram. The plugin can be fully configured using the config file

**Ru**

Этот плагин связывает чат майнкрафта и телеграма
при отправке смс в чат телеграм они будут клонироваться в майнкрафт также из майнкрафта в телеграм. Плагин полностью можно настроить с помнощю файла конфига
